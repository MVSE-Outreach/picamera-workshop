Digimakers Raspberry Pi Camera Workshop

1. Introduction

Hello there! Welcome to the Digimakers @-bristol picamera workshop! By the time you are finished here you'll know all about the Raspberry Pi camera module and how to control it using the Python programming language. Obviously, a little prior experience with Python is good but never fear, even if you have never experienced programming or the Raspberry Pi we will soon have you up'n at em in no time at all.

The main aim of the workshop is to give you a whistle stop tour of the picamera python module. This is essentially a nice and easy way of controlling the camera without having to use the raspistill and raspivid programs. This means we can be a lot clevererrerr in what we build!

The first thing you will need to do is to make sure that you have the picamera module installed on your Raspberry Pi. Make sure you are in the graphical environment for the pi (you might have to type startx when you first log in) and open up a terminal window. You can do this by clicking on the LXTerminal icon on your desktop.

If you have never used the camera before the first thing you will need to do is to enable it. Type raspiconfig and press the down arrow key until you reach "Enable Camera", press enter and enable the camera. You only need to do this if you have never used your picamera before!

Once you have done that exit the tool, remembering to reboot the pi so the install can finish. When you log back in you can open a terminal window again and type the following command to install the library: sudo apt-get install python-picamera That's it, you are all ready to go! Lets get programming!

2. Hello World! PiCamera!

For this first task we are going to make a very simple program that displays the camera's picture on your Pi for a few seconds and then makes it disappear. The code below shows you how to do this. Lazy people can download it here.

# import the two modules that we will need
import time
import picamera

# create our camera variable. We will be using this A LOT so remember it!
cam = picamera.PiCamera()

# start displaying the image preview on screen.
cam.start_preview()

# wait for ten seconds
time.sleep(10)

# close the preview window
cam.stop_preview()

# Always make sure you close the connection to your camera!
# Otherwise you won't be able to connect to it next time and
# you'll have to reboot!
cam.close()

If everything is installed and set up correctly you will see a big window fill your sceen with a lovely pretty picture of what your camera can see! If not, now would be the time to shout for help!
